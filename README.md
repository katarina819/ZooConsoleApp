# ZooConsoleApp

A simple console application written in C# that simulates a zoo management system.  
Users can add animals (like Lions, Elephants, Dogs, Mice, and Roosters), show animal info, and hear their sounds.

## Features

- Add new animals with name, age, and gender
- Display all animals with their details and sounds
- Input validation for name, age, and gender

## Technologies Used

- C# (.NET 9.0)
- Console application

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/katarina819/ZooConsoleApp.git
```
2. Open the solution file ZooApp.sln in Visual Studio.

3. Build the solution.

4. Run the project (usually F5 or Ctrl+F5).
